# ZK Prover Node

Clonne the repo.

```bash
git clone https://github.com/weavedb/weavedb.git
cd weavedb npm install && cd hb npm install && cd ../
```

Run.

```bash
yarn zkp --wallet ./HyperBEAM/.wallet.json --vid VID
```

Now, a zk-prover node should be running at [localhost:6365](http://localhost:6365).
