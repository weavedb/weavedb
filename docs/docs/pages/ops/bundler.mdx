# Httpsig Bundler Node

Clonne the repo.

```bash
git clone https://github.com/weavedb/weavedb.git
cd weavedb npm install && cd hb npm install && cd ../
```

Run.

```bash
yarn bundler --wallet ./HyperBEAM/.wallet.json --port 4001
```

Now, an Httpsig bundler node should be running at [localhost:4001](http://localhost:4001).
