# Scheduler Unit

Clonne the repo.

```bash
git clone https://github.com/weavedb/weavedb.git
cd weavedb npm install && cd hb npm install && cd ../
```

Run.

```bash
yarn su --wallet ./HyperBEAM/.wallet.json  --bundler "https://up.arweave.net:443/tx"
```

Now, an SU should be running at [localhost:4003](http://localhost:4003).
